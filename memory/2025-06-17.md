## 🔄 关键决策与变更

### 1. 项目定位修正
- **错误认知**: 最初将 OpenClaw 误解为 SDL2 游戏（含图形配置、手柄控制、游戏存档）
- **正确认知**: OpenClaw 是 Node.js AI 助手系统（类似 ChatGPT 私人助理）
- **依据**: 参考 `openclaw-cn-termux` 仓库和用户提供的真实安装流程

### 2. 安装方案变更
- **废弃方案**: 在 Termux 直接编译 C++/SDL2
- **正确方案**: Termux → proot-distro → Ubuntu → nvm → npm/pnpm
- **核心命令**:
  ```bash
  proot-distro install ubuntu  # 不指定版本号，默认最新
  nvm install --lts            # Node.js 24
  npm install -g openclaw-cn-termux@latest
  openclaw-cn-termux onboard  # 配置
  openclaw-cn-termux gateway  # 启动
  ```

### 4. 便捷脚本设计

**install-ubuntu.sh** (运行于 Termux):
- 自动更新包管理器
- 安装 proot-distro 等必要工具
- 配置存储权限
- 可选：设置国内镜像加速（清华镜像）
- 执行 `proot-distro install ubuntu`

**install-openclaw.sh** (运行于 Ubuntu):
- 更新系统，安装 sudo, nginx, git, build-essential, python3-pip 等
- 配置中文 locale (zh_CN.UTF-8)
- 检查/创建 openclaw 用户
- 安装 nvm + Node.js 24 LTS
- 可选择 npm 或 pnpm 安装 openclaw-cn-termux
- 验证安装并输出下一步指引

### 5. 版本号统一
- 所有文档中的 `proot-distro install ubuntu-22.04` 统一改为 `proot-distro install ubuntu`
- 理由：`proot-distro` 默认安装最新版 Ubuntu，无需指定版本
- 受影响文件：`installation.md`（开机自启部分）、`faq.md`（镜像加速说明）、`troubleshooting.md`（卸载命令）

## ⚠️ 注意事项

1. **不要基于游戏思维**: OpenClaw 是 AI 助手，不是游戏。不要添加图形配置、手柄映射等。
2. **proot-distro 版本**: 使用 `ubuntu` 而非 `ubuntu-22.04`，让 proot-distro 自动选择最新版。
3. **镜像加速**: 国内用户需要配置 `PROOT_DISTRO_MIRROR` 环境变量或使用代理。
4. **Node.js 版本**: 必须 ≥ 22，推荐 24 LTS。
5. **配置**: API Key 是必填项，首次运行 `openclaw-cn-termux onboard` 可交互配置。
6. **脚本可追溯**: 所有脚本均可通过 `--help` 查看帮助，并可在编辑器中检查源码。
